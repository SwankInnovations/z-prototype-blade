New in this revision:

  • Added text shadowing. Turns out Mephisto uses the CSS3 'text-shadow' attribute that Safari accepts but no other browser does (which is why I didn't see it in FF). So, I used a prototype based script to add the shadowing for all of our tested browsers (with one styling bug -- IE doesn't currently render the text shadow on 'Metadata'. Good enough for now).
  • Created the Metadata section and even added a scriptaculous effect on opening/closing. I had no idea what you were thinking for styling in there so I guessed and stole from the existing Radiant design.
  • Created the 'view site' button. I enhanced the behavior using a modal window -- let me know what you think. (No, I don't like the visual design either, it's the conceptual design I'm talking about, here).
  • Added a mouseover image to the secondary navigation links. I'm not sure I like it better than the previous underline-on hover, though. That simplicity was nice. I could, however, see setting up links on the right hand side of the secondary nav for contextual things like 'help' or 'render page' and they could use a style like that.
  • Played more with the whole translucency bit (IE6 still doesn't get it, though). It's pretty cool. Check out the tabs and metadata area when you change the background colors and images!
